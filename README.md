Página pessoal de portfólio desenvolvida com **HTML5** e **CSS3** puro, sem frameworks ou bibliotecas externas. Apresenta minhas habilidades, tecnologias que utilizo e links para contato.

## ✨ Funcionalidades

- 🖼️ Exibição de ícones das tecnologias utilizadas via [Devicons](https://devicon.dev/) e [Simple Icons](https://simpleicons.org/)
- 🔗 Links diretos para LinkedIn, GitHub e Email
- 💜 Animação de pulso nos cards de redes sociais
- 📱 Layout em grid responsivo
- 🌙 Tema escuro com gradiente roxo

## 🚀 Como Rodar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/TaianeAlbuquerqueDev/portfolio.git
```

2. Acesse a pasta do projeto:
```bash
cd portfolio
```

3. Abra o arquivo `index.html` no navegador:
```bash
# No Windows
start index.html

# No Mac
open index.html

# No Linux
xdg-open index.html
```

> Não requer instalação de dependências ou servidor — abre direto no navegador! ✅
